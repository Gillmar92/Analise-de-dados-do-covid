# Analise-de-dados-do-covid

Com base nos dados de covid presentes em https://github.com/turicas/covid19-br/blob/master/api.md#caso_full, fez-se o carregamento, seleção e análise de dados referentes
às cidades de: São Paulo, Rio de Janeiro, Santana de Parnaíba, Feira de Santana e Barra Mansa.
Neste repositório é possível encontrar os programas em python necessários para a criação do banco de dados, bem como para o carregamento das informações no banco. Além disso,
também é possível conferir o diagrama do banco de dados criado e um relatório com a análise de dados para as 5 cidades selecionadas.
